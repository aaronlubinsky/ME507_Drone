\doxysubsection{Detailed Description}
ESC (Electronic Speed Controller) driver for quadcopter motor control. 

\begin{DoxyAuthor}{Author}
Aaron Lubinsky 
\end{DoxyAuthor}
\begin{DoxyVersion}{Version}
1.\+0 
\end{DoxyVersion}
\begin{DoxyDate}{Date}
2025
\end{DoxyDate}
This driver implements PID-\/based motor control for a quadcopter ESC system. It provides functionality to arm ESCs and update motor speeds based on roll, pitch, and yaw control inputs using PID control algorithms.

\DoxyHorRuler{0}
 \hypertarget{_e_s_c_8c_autotoc_md3}{}\doxysubsection{\texorpdfstring{\#\#\#\#\# How to use this driver \#\#\#\#\#}{\#\#\#\#\# How to use this driver \#\#\#\#\#}}\label{_e_s_c_8c_autotoc_md3}

\begin{DoxyEnumerate}
\item Call \doxylink{_e_s_c_8c_a54963d29d4caf06cd558dd118520737f}{arm\+ESC()} to initialize and arm all ESC motors
\item Call \doxylink{_e_s_c_8c_adf18312a6633c434fac75e4bf7f91a15}{update\+\_\+\+Motors()} periodically to update motor speeds based on PID control
\item Ensure all external variables (PID constants, setpoints, etc.) are properly configured
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
This driver requires STM32 HAL library and Timer 3 configured for PWM output 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Motor safety\+: Always ensure proper calibration of motor offsets before flight 
\end{DoxyWarning}

Arms all ESC motors by sending initialization sequence. 

Arms the ESCs with a minimum throttle signal.

This function sends the proper PWM signals to arm all four ESC motors. It starts PWM generation on all channels and sends a specific pulse width (approximately 1000μs) to arm the ESCs. The function continues until the roll\+\_\+set value reaches a threshold, indicating the system is ready.

\begin{DoxyNote}{Note}
This function blocks execution until arming is complete 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Ensure motors are properly secured before calling this function

void update\+\_\+\+Motors (\begin{DoxyParamCaption}\item[{void}]{}{}\end{DoxyParamCaption})}



Updates all motor speeds based on PID control calculations. 

Updates the motor speeds based on control inputs.

This function performs the main control loop operation\+:
\begin{DoxyEnumerate}
\item Calculates PID errors for roll, pitch, and yaw axes
\item Computes PID control efforts for each axis
\item Mixes control efforts to determine individual motor speeds
\item Applies safety limits and updates PWM outputs
\end{DoxyEnumerate}

The motor mixing follows a standard quadcopter X-\/configuration\+:
\begin{DoxyItemize}
\item Motor A (front-\/right)\+: +pitch, -\/roll, -\/yaw
\item Motor B (rear-\/right)\+: -\/pitch, -\/roll, +yaw
\item Motor C (rear-\/left)\+: -\/pitch, +roll, -\/yaw
\item Motor D (front-\/left)\+: +pitch, +roll, +yaw
\end{DoxyItemize}

\begin{DoxyNote}{Note}
PWM range\+: 960 (0\% throttle) to 2000 (100\% throttle) 

For safety, maximum throttle is limited to 1500 (≈50\%) 

Function should be called at regular intervals (typically 1k\+Hz)
\end{DoxyNote}
